# v0.1.0 release checklist

- [ ] `make verify` passes.
- [ ] `make demo` generates the checked-in report assets from a bundled fixture.
- [ ] `make package` creates wheel, sdist, report bundle, and SHA256SUMS.
- [ ] `make release-check` validates version, changelog, clean tree, author, secrets, and artifacts.
- [ ] GitHub CI is green on `main`.
- [ ] Annotated `v0.1.0` tag and GitHub Release contain the release assets.
